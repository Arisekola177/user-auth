
*User Authentication Application*

*Overview*

This application provides a secure user authentication system, built with Next.js, Prisma, MongoDB, and Tailwind CSS. It allows users to register, login, and activate their accounts, with email verification and also allow users to reset their password if forgotten.

*Features*

- User registration with email verification
- Login functionality with session management
- Account activation via email link
- Redirects to home page for authenticated users
- Redirects to login page for unauthenticated users
- Password Reset

*Technologies Used*

- Next.js (React framework)
- Prisma (ORM for MongoDB)
- MongoDB (NoSQL database)
- Tailwind CSS (styling framework)

*Getting Started*

1. Clone the repository: `git clone https://github.com/Arisekola177/user-auth.git`

2. Install dependencies: `npm install` or `yarn install`

3. Start the development server: `npm run dev` or `yarn dev`

4. Open the application in your browser: `http://localhost:3000`

*Deployment*

This application is ready for deployment on Vercel or other Next.js compatible platforms.
Live link:https://user-auth-ivory-six.vercel.app/
